# printf - Custom printf Implementation

## Project Description
This project is a custom implementation of the printf function in C, created as part of the Holberton School curriculum. The function `_printf` produces formatted output according to a specified format string.

## Requirements
- **Compilation**: Ubuntu 20.04 LTS, gcc with flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- **Style**: Betty coding style compliant
- **Restrictions**: No global variables, max 5 functions per file
- **Authorized functions**: `write`, `malloc`, `free`, `va_start`, `va_end`, `va_copy`, `va_arg`

## Features Implemented

### Task 0: Basic Conversion Specifiers
- `%c` - Print a single character
- `%s` - Print a string
- `%%` - Print a literal percent sign

### Task 1: Integer Conversion Specifiers  
- `%d` - Print signed decimal integer
- `%i` - Print signed integer (same as %d)

### Task 2: Documentation
- Created man page: `man_3_printf`
